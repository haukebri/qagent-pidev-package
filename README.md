# qagent-pidev-package

Pi package for QAgent.

## Tools

- `hello_world`: smoke-tests Pi model/auth access.
- `qagent_run`: runs the installed `@qagent/cli` command with one URL and one
  natural-language verification goal.

## Local Install

```bash
npm install
pi install ./
```

For local path installs, `npm install` provides `node_modules/.bin/qagent`.
Alternatively, install QAgent globally so `qagent` is on `PATH`.
